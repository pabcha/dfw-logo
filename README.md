# DFW Logo Web Component

Un web component que muestra "DataFlow" en un div.

## Instalación

```bash
npm install
```

## Compilación

```bash
npm run build
```

## Desarrollo

```bash
npm run dev
```

## Uso

Abre `index.html` en tu navegador o ejecuta:

```bash
npm start
```

Luego abre `http://localhost:8000` en tu navegador.
